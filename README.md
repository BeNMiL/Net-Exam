# CompTIA Network+ N10-009 Practice Exam Platform

## Prerequisites
- **Node.js** (version 18 or higher) — Download from https://nodejs.org

## Setup (one time)

1. Open this folder in VS Code
2. Open the terminal in VS Code (`Ctrl + ~` or `Cmd + ~`)
3. Run these two commands:

```bash
npm install
npm run dev
```

4. Your browser will open automatically to `http://localhost:5173`
5. The app is now running fullscreen in your browser!

## Daily Use

Every time you want to study, just open the terminal and run:

```bash
npm run dev
```

Press `Ctrl + C` in the terminal to stop it when done.

## Tips
- Your progress, scores, and flagged questions are saved in your browser automatically
- Press F11 in your browser for true fullscreen
- Works best in Chrome or Edge
